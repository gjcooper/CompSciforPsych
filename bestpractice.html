def buildText(self):
    final_str = ''
    for section in cycle('1234'):
        if len(final_str) < self.textlength:
            final_str += section
        else:
            if self.reverseText:
                return reversed(final_str)
            else:
                return final_str

  • DON'T DO: Can you look over my code/experiment

  • Highlight any specific aspects you want checked
